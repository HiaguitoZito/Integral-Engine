In order for you to compile:

1° You need MinGW installed!
for WINDOWS:
* https://sourceforge.net/projects/mingw/
for LINUX:
* https://www.mingw-w64.org/downloads/

2° You need to drag the $(PROJECTPATH)/lib/libglfw3.a to $(MINGWPATH)/lib

3° Open the CMD in the root project folder and type: "makefile".
